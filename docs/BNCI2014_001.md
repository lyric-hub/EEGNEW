# BNCI2014_001 Dataset

> **Source**: BCI Competition IV - Dataset IIa  
> **Leaderboard**: [PapersWithCode](https://paperswithcode.com/dataset/bnci2014-001-moabb-1)

## Overview

| Property | Value |
|----------|-------|
| **Subjects** | 9 |
| **Channels** | 22 (Ag/AgCl electrodes) |
| **Classes** | 4 |
| **Trials per class** | 144 |
| **Trial length** | 4 seconds |
| **Sampling rate** | 250 Hz |
| **Sessions** | 2 (on different days) |
| **Runs per session** | 6 |
| **Total trials** | 62,208 |

## Motor Imagery Classes

| Class | Task |
|-------|------|
| 1 | Left hand imagination |
| 2 | Right hand imagination |
| 3 | Both feet imagination |
| 4 | Tongue imagination |

## Experimental Protocol

1. **t = 0s**: Fixation cross appears + acoustic warning tone
2. **t = 2s**: Cue arrow appears (direction indicates class)
3. **t = 3.25s**: Cue disappears (subject continues imagery)
4. **t = 6s**: Fixation cross disappears (trial ends)

**No feedback** was provided during the trials.

## Recording Setup

- **Electrodes**: 22 Ag/AgCl electrodes (3.5 cm inter-electrode distance)
- **Reference**: Left mastoid (monopolar recording)
- **Ground**: Right mastoid
- **Bandpass filter**: 0.5 Hz - 100 Hz
- **Notch filter**: 50 Hz (line noise suppression)
- **Amplifier sensitivity**: 100 μV

## Session Structure

```
Session (1 of 2)
├── Run 1: 48 trials (12 per class)
├── Run 2: 48 trials (12 per class)
├── Run 3: 48 trials (12 per class)
├── Run 4: 48 trials (12 per class)
├── Run 5: 48 trials (12 per class)
└── Run 6: 48 trials (12 per class)
    = 288 trials per session
```

## Usage in MOABB

```python
from moabb.datasets import BNCI2014_001

dataset = BNCI2014_001()

# Get data for all subjects
data = dataset.get_data()

# Access specific subject
subject_1_data = dataset.get_data(subjects=[1])
```
